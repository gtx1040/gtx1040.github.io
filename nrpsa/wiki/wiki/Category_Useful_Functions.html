The Useful Functions category contains Lua functions contributed by our community. Feel free to use them in your scripts but also to add new useful functions or optimize existing ones.

Be geared to the style of NRP's server and client functions. A good example of how a useful function page should look like you can get here. Don't forget to add the function to the Useful functions category with {{Useful Function}}! Please don't delete or modify functions of others for no reason.

By the way: The idea came from an outdated topic on the forums.
