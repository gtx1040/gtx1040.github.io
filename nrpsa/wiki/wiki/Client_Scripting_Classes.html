This page lists all the client-side scripting classes that are implemented in the 1.4 version of Nexus RP. It mirrors the procedural client functions. For information on how to enable OOP in 1.4 please refer to Meta.xml.

Classes can be instantiated by either calling the create method, when applicable, or by calling the class. For example: Vehicle(...) is the same as Vehicle.create(...).
