Basic IRC module (bIRC) provides a fully functional server IRC echo for NRP:SA. It is currently available only for Windows servers on NRP:SA 1.0 (other versions untested).

Installation

Uncompress the file ml_birc.dll into your server\mods\deathmatch\modules\ directory.

nrpserver.conf

Add the following line in your nrpserver.conf: 

<module src="ml_birc" />
